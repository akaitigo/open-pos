> 調査日: 2026-03-28

## 3. open-pos への示唆

### 3.1 全POSに共通する必須機能（Must-Have）

以下は調査した全7システムが標準搭載している機能であり、open-pos v1.0 で必須:

1. **基本レジ会計**（商品登録、バーコードスキャン、カスタム商品）
2. **割引処理**（金額/パーセント）
3. **複数税率対応**
4. **返品・返金処理**
5. **レジ締め/精算**
6. **レシート印刷/発行**
7. **リアルタイム在庫追跡**（販売連動自動更新）
8. **商品カテゴリ/バリエーション管理**
9. **顧客プロファイル・購買履歴**
10. **スタッフアカウント・権限管理**
11. **売上レポート**（日別/商品別/時間帯別/スタッフ別）
12. **税金レポート**
13. **CSV入出力**
14. **オフライン販売**
15. **REST API**

### 3.2 差別化に有効な機能（Should-Have）

多くのシステムが有料アドオンとして提供しており、open-pos がOSSとして標準搭載すれば差別化要因になる機能:

1. **店舗間在庫移動**（Square/Shopify/スマレジは有料）
2. **ロイヤルティ/ポイントプログラム**（ほぼ全社が有料アドオン）
3. **ロス管理**（スマレジのみ有料提供、他は未対応）
4. **棚卸し機能**（多くが有料）
5. **複数店舗パフォーマンス比較**（多くが有料）
6. **BOPIS（オンライン購入→店舗受取）**
7. **メールマーケティング**
8. **カスタムレポート**

### 3.3 日本市場向け必須機能

open-pos が日本市場をターゲットにする場合、以下は必須:

1. **軽減税率対応**（8%/10% 区分）
2. **インボイス制度対応**（適格請求書発行）
3. **電子帳簿保存法対応**
4. **日本の会計ソフト連携**（freee/弥生）
5. **QRコード決済対応**（PayPay等）
6. **電子マネー対応**（Suica/PASMO等）
7. **免税販売対応**
