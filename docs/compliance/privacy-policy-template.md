# プライバシーポリシー（テンプレート）

> **注意**: 本テンプレートは法務部/外部弁護士のレビュー後に正式版として公開すること。

## 1. 収集する個人情報

| データ | 収集対象 | 利用目的 |
|-------|---------|---------|
| 氏名 | スタッフ/顧客 | アカウント管理、レシート記載 |
| メールアドレス | スタッフ/顧客 | アカウント通知、領収書送付 |
| 電話番号 | 顧客（任意） | 予約確認連絡 |
| PIN ハッシュ | スタッフ | POS端末認証 |
| 購買履歴 | 顧客 | ポイント管理、サービス改善 |
| IPアドレス | 同意記録時 | 同意取得の証跡 |

## 2. 利用目的

1. POS取引の処理・記録
2. ポイント・ロイヤリティプログラムの運営
3. レシート・領収書の発行
4. 在庫管理・発注
5. 売上分析・レポート生成
6. システムの運用・保守・改善
7. 法令に基づく義務の履行

## 3. 第三者提供

原則として本人の同意なく第三者に提供しません。
ただし以下の場合を除きます：
- 法令に基づく場合
- 人の生命・身体・財産の保護に必要な場合

## 4. 安全管理措置

- データの暗号化（通信: TLS 1.3、保存: AES-256）
- アクセス制御（RBAC: Owner/Manager/Cashier）
- マルチテナント分離（Hibernate Filter）
- 監査ログの記録
- bcrypt によるPINハッシュ化

## 5. データの保持期間

| データ種別 | 保持期間 | 根拠 |
|-----------|---------|------|
| 取引データ | 7年 | 電子帳簿保存法 |
| 監査ログ | 7年 | 電子帳簿保存法 |
| 顧客PII | 退会後1年 | 個人情報保護法 |
| スタッフPII | 退職後3年 | 労働基準法 |

## 6. 開示・訂正・削除の請求

個人情報の開示・訂正・削除を希望する場合は、
管理者（OWNER ロール）を通じて GDPR エンドポイントで対応。
